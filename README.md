# Digi Crawler

![NodeJS Version](https://img.shields.io/badge/nodejs-16.14.2-blue)

DigiByte Crawler is a node scaner for the DigiByte blockchain.

## Development 🛠️

```sh
git clone https://github.com/RenzoDD/digi-crawler
cd digi-crawler
npm install
```
