DigiByte Crawler is a node scaner for the DigiByte blockchain.

## Development 🛠️

```sh
git clone https://github.com/RenzoDD/digi-nodes
cd digi-nodes
npm install
```
```sh
cd db
type *.sql > script
```

Run the script into your MySQL database
